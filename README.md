💼 AI-Powered Job Consultancy Platform

A modern job consultancy web platform designed to streamline recruitment workflows for both job seekers and recruiters.
The platform enables companies to post job opportunities while allowing candidates to build professional resumes using customizable templates and themes.

This system simplifies hiring by combining job listings, resume building tools, and candidate management in one unified platform.

# Key Features

Job Listings

Recruiters can create and manage job postings.

Candidates can browse available opportunities.

Organized job categories for easy navigation.


Resume Builder

Interactive resume builder for job seekers.

Multiple professional resume templates.

Customizable themes and layouts.

Download-ready resumes for applications.


Candidate Management

Store and manage candidate profiles.

Resume upload and storage.

Organized candidate data for recruiters.


User-Friendly Interface

Clean and responsive UI.

Easy navigation for both recruiters and applicants.

Install dependencies:

npm install

npm run dev
